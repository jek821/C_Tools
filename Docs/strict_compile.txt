gcc -std=c11 -D_POSIX_C_SOURCE=200809L -Wall -Wextra -Wpedantic -Wshadow -Wstrict-prototypes -Wmissing-prototypes -Wconversion -Werror -o <output> <target>

Note: -D_POSIX_C_SOURCE=200809L is required. Under strict -std=c11 the POSIX
API (getopt, optind, ...) is hidden behind feature-test macros, so without it
the build fails with "implicit declaration of getopt" / "optind undeclared".
