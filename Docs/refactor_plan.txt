REFACTOR PLAN: jek_ls + project-wide cleanup
============================================

Reference: jek_cat and jek_wc are the gold standard — clean POSIX syscalls,
full error handling, a test.sh suite, a header with prototypes, and a Makefile
that builds under the strict standard (Docs/strict_compile.txt) and has a
`test` target.

STATUS: This plan is complete.
  - The jek_wc section was completed and removed earlier.
  - PROJECT-WIDE CLEANUP is done: .gitignore covers all build artifacts
    (jek_*/jek* with source/header/script un-ignores; stale jek_*/main and
    jek_*/test patterns), no binaries are tracked, no stale binaries remain on
    disk, and the root Makefile's all/install/uninstall/clean loops already
    use `|| exit 1`.
  - jek_ls is now at the gold-standard bar: jek_ls.c was renamed to jekls.c;
    the Makefile uses the strict CFLAGS and has a `test` target; jekls.h
    carries a real prototype (list_dir); test.sh mirrors jek_cat/test.sh.
  - SOURCE FIXES (jekls.c) done: getopt -a flag, optional path argument,
    dotfiles (incl. "." / "..") hidden unless -a, directory names colored and
    reset, returns 1 on error / 0 on success, debug mtime block removed. The
    leaking malloc'd struct stat and the unchecked fd_to_buff()/stat() were
    deleted outright rather than patched, since removing the debug time output
    left them with no caller; directory detection uses dirent d_type. Unused
    <fcntl.h>/<time.h>/<sys/stat.h>/<stdlib.h> includes removed.

VERIFICATION (passing):
  make clean && make && make test   # all three tools build + tests pass
                                     # (jek_cat 8/8, jek_wc 26/26, jek_ls 10/10)

  Manual spot-checks:

    ./jek_ls/jekls -a       # also shows dotfiles
